# Running Locally

1. Clone the repository:

```bash
git clone https://github.com/blackzarifa/pokedex-next.js.git
cd pokedex-next.js
```

2. Install dependencies:

```bash
# Using npm
npm install

# Using pnpm
pnpm install

# Using yarn
yarn install
```

3. Start the development server:

```bash
# Using npm
npm run dev

# Using pnpm
pnpm dev

# Using yarn
yarn dev
```

4. Open [http://localhost:3000](http://localhost:3000) in your browser
